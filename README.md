# file-assoc-in

Utility for updating clojure data files containing maps (such as leiningen profiles), without losing comments and lexical layout.

## Usage

`lein run "C:\Users\Dave\.lein\profiles.clj" "[:user :java-cmd]" "c:\jdk\bin\java.exe"`
